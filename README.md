# EmEnvSet - Embedded Environment Setup System

Simple script for setting up an embedded development environment on a freshly-installed Linux system

## Getting Started

### Prerequisites

* Linux OS
    * Ubuntu/Debian
* Internet connection

### Usage

```bash

chmod +x em-env-set.sh

./em-env-set.sh
```
