# DEnvSet - Development ENVironment SETup Framework

Simple framework for setting up a development environment on a freshly-installed Linux system

## Getting Started

### Prerequisites

* Linux OS
    * Ubuntu/Debian
* Internet connection

### Usage

```bash

chmod +x denvset.sh

./denvset.sh
```
